# My Portfolio Web App

A portfolio web app made in Next.js.

## Features

- Responsive design
- Interactive UI

## Technology Stack

- Next.js
- Tailwind CSS
- React

## Installation and Setup
Feel free to clone my repo:
```bash
git clone 'https://github.com/sonyerg/portfolio-nextjs-app'
cd [portfolio-nextjs-app]
npm install
npm run dev
```
